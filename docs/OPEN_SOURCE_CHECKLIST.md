# Open Source Checklist

## Engineering

- [ ] package install works from a clean environment
- [ ] README launch steps are accurate
- [ ] manifests remain backward-compatible or have a migration note
- [ ] CPU-only workflow tested on a real batch
- [ ] CUDA workflow tested on a real batch

## Product

- [ ] screenshots captured
- [ ] alpha roadmap written
- [ ] first issue templates prepared

## Legal and compliance

- [ ] license selected
- [ ] third-party notices reviewed
- [ ] model source terms reviewed
- [ ] `ffmpeg` redistribution decision made

## Release hygiene

- [ ] version bumped
- [ ] changelog drafted
- [ ] tag and release notes prepared
